# crabygen

CLI tool for [Craby](https://github.com/leegeunhyeok/craby)

## Installation

```bash
npm install --save-dev crabygen
# or
pnpm add -D crabygen
# or
yarn add -D crabygen
```

## Documentation

Visit [https://craby.rs](https://craby.rs) for full documentation.
